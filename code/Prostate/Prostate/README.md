# prostate_project   

This is the code for prostate segmentation.     
Using supervised learning, segment PZ or TZ.   

# Requirements       
- Pytorch version >=0.4.1.     
- Python == 3.6     
- TensorBoardX    
- Some basic python packages such as Numpy...
